# 🧠 Prompt: Generate a Modern SaaS Landing Page for PK-Manager

Create a **modern, high-quality SaaS landing page** for a product called **PK-Manager (Personal Knowledge Manager)**.

The design should feel like a mix of **Notion, Linear, and modern AI tools** — clean, minimal, fast, and professional.

---

## 🎯 Goal of the Page

- Welcome users
- Explain the product clearly
- Build trust
- Encourage users to **Sign In / Get Started**
- Serve as an **intro before entering the dashboard**

---

## 🧱 Sections to Include

### 1. 🔝 Hero Section (First Impression)

- Headline: Strong, bold, clear
  - Example: _"Organize Your Mind. Execute Your Ideas."_

- Subtext: Explain PK-Manager in 1–2 lines
- CTA Buttons:
  - "Get Started"
  - "Go to Dashboard"

- Visual:
  - App preview (notes, tasks, dashboard UI mock)

---

### 2. 🧠 What is PK-Manager

- Short explanation of the platform
- Emphasize:
  - Notes
  - Tasks
  - Goals
  - Journal

- Show it as a **connected system**, not separate tools

---

### 3. ⚡ Features Section

Display 4–6 feature cards:

- 📝 Smart Notes (markdown, linking)
- 🔗 Connected Thinking (link notes, tasks, goals)
- ✅ Task Execution (turn ideas into action)
- 📓 Daily Journal (track progress)
- 🎯 Goal Tracking (long-term focus)
- 🔍 Fast Search (optional)

Each card should have:

- Icon
- Title
- Short description

---

### 4. 💡 How It Works (Simple Flow)

Show steps like:

1. Capture ideas (Notes)
2. Connect them (Tags & Links)
3. Turn into tasks
4. Track progress (Journal & Goals)

Use a clean horizontal or vertical step layout

---

### 5. 🧑‍💻 Who It’s For

- Developers learning new skills
- Students organizing knowledge
- Creators managing ideas
- Anyone building a “second brain”

---

### 6. ⭐ Testimonials (Fake but realistic)

Include 2–3 testimonials like:

> "PK-Manager completely changed how I learn and track my progress."

> "It feels like Notion, but built for focused thinking."

Make them look real with:

- Name
- Role (e.g., Frontend Developer)

---

### 7. 🚀 Call to Action (Final Push)

- Strong closing message
- Button: "Start Building Your Second Brain"

---

### 8. 🦶 Footer

- Links:
  - About
  - Features
  - Contact

- Social icons (optional)
- Copyright

---

## 🎨 Design Requirements

- Use **Next.js + Tailwind CSS**
- Clean spacing, modern typography
- Responsive (mobile + desktop)
- Use subtle animations (hover, fade-in)
- Use icons (Lucide React preferred)

---

## 🧠 UX Details

- Sticky navbar
- Smooth scrolling
- Highlight CTA buttons
- Fast loading

---

## ⚙️ Extra (Optional but Powerful)

- Dark mode toggle
- Command palette (Cmd + K)
- Animated UI preview

---

## 📌 Output Format

- Provide full React/Next.js component
- Clean, reusable sections
- Well-structured code

---

## 🚨 Important

Do NOT make it look basic.

This should look like a **real SaaS product landing page** that can impress recruiters and clients.

Focus on:

- Simplicity
- Clarity
- Professional feel
