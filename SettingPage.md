# ⚙️ SETTINGS PAGE — PKM SYSTEM CONTROL CENTER

You are implementing the Settings Page for a Personal Knowledge Management (PKM) system.

This page is NOT just a preferences screen.

It is the **SYSTEM BEHAVIOR CONTROL LAYER** that defines how the entire AI + PKM system operates.

---

# 🎯 CORE PURPOSE

The Settings Page exists to:
- Control AI behavior across the system
- Define automation rules for Tasks, Dreams, Inbox, and Insights
- Manage user preferences for productivity behavior
- Adjust system strictness, intelligence, and automation levels

---

# 🧠 SYSTEM CONTROL AREAS

## 1. AI BEHAVIOR SETTINGS (CRITICAL)

Control how the AI behaves globally:

- AI strictness level:
  - low → flexible suggestions
  - medium → balanced guidance
  - high → strict execution enforcement

- AI proactiveness:
  - passive → only responds when asked
  - active → suggests tasks, insights, improvements
  - autonomous → auto-generates tasks and insights

- AI reasoning depth:
  - fast → quick responses
  - balanced → normal reasoning
  - deep → detailed analysis across PKM

---

## 2. AUTOMATION SETTINGS

Control system automation rules:

- Auto-task generation from Dreams (ON/OFF)
- Auto-linking in Knowledge Graph (ON/OFF)
- Auto-insight generation frequency:
  - real-time
  - hourly
  - daily
- Inbox auto-routing sensitivity:
  - strict classification
  - flexible classification

---

## 3. TASK SYSTEM BEHAVIOR

- Task prioritization mode:
  - manual
  - AI-assisted
  - fully automated

- Deadline enforcement:
  - soft (reminders only)
  - medium (alerts + escalation)
  - strict (ledger penalty tracking)

---

## 4. LEDGER ACCOUNTABILITY SETTINGS

Define how strict the system tracks execution:

- strictness level:
  - soft → motivational tracking
  - balanced → standard tracking
  - strict → failure-sensitive accountability system

- failure visibility:
  - hidden
  - visible to user only
  - highlighted in dashboard & insights

---

## 5. DREAM SYSTEM SETTINGS

Control how dreams generate structure:

- auto project generation from dreams (ON/OFF)
- auto task breakdown from dreams (ON/OFF)
- dream progress sensitivity (low/medium/high)

---

## 6. NOTIFICATION SETTINGS

- task reminders
- focus session alerts
- daily insight summaries
- missed journal reminders

---

# 🤖 AI SETTINGS ENGINE RULES

The system must:
- adapt AI behavior dynamically based on settings
- enforce strictness rules consistently across all modules
- override default AI behavior when settings conflict
- ensure all automation respects user-defined boundaries

---

# 🧠 BEHAVIOR RULE

Settings act as the **highest authority layer** in the system.

If a setting conflicts with AI suggestion logic:
→ SETTINGS ALWAYS WIN

---

The Settings Page must function as:

> “The control panel of the user’s entire cognitive operating system”
